No Long Functions
======================

A [CoffeeLint][coffeelint] rule that prohibits long functions.

[coffeelint]: http://www.coffeelint.org/

##Usage
Insert these lines into your coffeelint config:
```
 "no_long_functions": {
     "module": "coffeelint-no-long-functions",
     "level": "error",
     "value": 20
   }
```
